# DORA (Regulation (EU) 2022/2554) ruleset

This ruleset is for building a structured, auditable checklist for **Digital Operational Resilience Act (DORA)**.

## Key note: applicability depends on entity type
DORA obligations vary depending on the **financial entity** category and whether the assessed party is:
- a financial entity (bank, payment institution, investment firm, etc.)
- an ICT third‑party service provider

We will model this via `entity_profile` parameters (see `entity_profile.schema.yaml`).
